# Non-Contact Friction Simulator

A simple physics-based simulation that models non-contact friction (drag force) using a velocity-dependent equation.

---

## 📐 Equation Used

F = k · v²

---

## Terms

- **F** → Friction (drag) force opposing motion  
- **v** → Velocity of the object  
- **k** → Constant representing properties of the medium (e.g., air, water)

---

## Concept

This model is derived from the real-world drag equation, where drag force increases with the square of velocity.

As velocity increases:
- More particles collide with the object
- Each collision transfers more momentum

This results in a rapid increase in resistance (drag force).
